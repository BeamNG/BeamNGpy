beamng_ros2.publishers.viz module

class beamng_ros2.publishers.viz.VehicleVisualizer(scale: Tuple[float, float, float], offset: ndarray, color: Tuple[float, float, float, float], arrow_color: Tuple[float, float, float, float])

Bases: VehiclePublisher

Publishes information about the BeamNG vehicle position and rotation as a marker in the form of MarkerArray messages.

Parameters:

  • scale (Tuple[float, float, float]) – Scale of the vehicle in meters in the ROS2 coordinate system.

  • offset (ndarray) – Offset between the position that BeamNG provided (the main node of the vehicle) and the position we visualize (the geometric center).

  • color (Tuple[float, float, float, float]) – The color of the vehicle.

  • arrow_color (Tuple[float, float, float, float]) – The color of the arrow that points in the direction of the vehicle.
